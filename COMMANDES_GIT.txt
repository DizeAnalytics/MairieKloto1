=== COMMANDES GIT POUR POUSSER LE PROJET SUR GITHUB ===

1. Créer le dépôt sur GitHub d'abord :
   - Aller sur https://github.com/DizeAnalytics
   - Cliquer sur "New repository"
   - Nom : MairieKloto1
   - NE PAS cocher "Initialize with README"
   - Cliquer sur "Create repository"

2. Ouvrir Git Bash dans le dossier du projet et exécuter ces commandes une par une :

git config --global user.email "dizeanalytics@gmail.com"

git config --global user.name "DizeAnalytics"

git init

git add .

git commit -m "Initial commit: Plateforme web Mairie de Kloto 1"

git remote add origin https://github.com/DizeAnalytics/MairieKloto1.git

git branch -M main

git push -u origin main

3. Lors de "git push", utiliser :
   - Username : dizeanalytics@gmail.com
   - Password : Votre Personal Access Token GitHub (pas le mot de passe)
   
   Pour créer un token : https://github.com/settings/tokens
   → Generate new token (classic) → cocher "repo" → Generate

=== FIN DES COMMANDES ===
